# Medical-Expense-Regression
Predicts medical insurance expenses using regression algorithms (Linear Regression) with features like age, sex, BMI, children, smoking status, and region. Includes data cleaning, model training, evaluation, and example predictions in a Jupyter notebook.

Note- For Prediction 
Categorical features have been encoded numerically as follows:

Original feature       Categorical Value     Numeric Input
Sex                    Female                1 
                       Male                  0  

Smoker                 No                    1 
                       Yes                   0

Region                  Southeast             0
                       Southwest             1 
                       Northwest             3
                       Northeast             3
## Dependencies

- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
